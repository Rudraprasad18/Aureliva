# Aureliva 🛍️

Aureliva is a modern e-commerce web application designed to provide users with a smooth online shopping experience. The project includes product browsing, user authentication, order tracking, checkout, invoice generation, and an admin dashboard.

## Features

- User Login System
- Product Catalog
- Shopping Cart
- Checkout Page
- Invoice Generation
- Order Tracking
- Admin Dashboard
- Responsive Design
- Interactive User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Rudraprasad18/Aureliva.git
```

2. Navigate to the project folder:

```bash
cd Aureliva
```

3. Install dependencies:

```bash
npm install
```

4. Start the server:

```bash
node server.js
```
